Stateless Architecture

Permanent QR Codes.
Zero Dependencies.

A lightweight SaaS-style platform to generate, customize, and embed permanent, self-contained QR codes directly in your browser. No external APIs. No expiration.

Launch CosmoEncode View Docs

The Vision

Providing a simple, reliable, and sustainable QR service. Unlike tools generating temporary links, CosmoEncode produces permanent embeds that remain functional indefinitely.

Permanent

Generates as Base64 encoded images. Embedded directly into HTML instead of loaded from servers.

Offline Capable

Fully self-contained and works completely offline. Zero external API calls required to render.

Sustainable

Avoids repetitive server requests. Reduces network traffic and ensures long-term reliability.

Exportable

Download in PNG for web/print, or SVG for infinite scaling without loss of quality.

Client-Side Architecture

CosmoEncode uses a fully client-side architecture built on HTML5, CSS3, and JavaScript. No backend processing or database storage is required, making it perfect for static hosting environments like GitHub Pages.

  • 1 User enters URL or text.
  • 2 QR is generated locally in the browser.
  • 3 Canvas renders the QR image.
  • 4 Image converted to Base64 string.
  • 5 User downloads or embeds the snippet.

Embed the Code

Because the QR image is embedded using Base64 encoding, the QR code remains permanent and independent of any servers.

index.html 
<!-- Generated with CosmoEncode -->
<div style="text-align:center;">
  <img src="data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAA..." 
       alt="Permanent QR Code" 
       width="200">
  <p style="font-family: monospace;">Scan to visit</p>
</div>
